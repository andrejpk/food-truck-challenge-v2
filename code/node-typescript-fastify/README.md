# Node TypeScript Fastify Starter

Starter shell for the Food Truck Challenge using Node.js, TypeScript, and Fastify.

## Stack

- **Node.js 18+**
- **TypeScript** — strict mode
- **Fastify** — high-performance REST API framework
- **Vitest** — fast test runner
- **tsx** — TypeScript execution for development

## Getting Started

```bash
npm install
npm run dev
```

## Running Tests

```bash
npm test
```
