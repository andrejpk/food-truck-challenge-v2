# .NET 10 ASP.NET Core Starter

Starter shell for the Food Truck Challenge using .NET 10 and ASP.NET Core.

## Stack

- **.NET 10** — latest LTS
- **ASP.NET Core Minimal APIs** — lightweight endpoint mapping in `Api.cs`
- **ASP.NET Core Controllers** — `HelloController.cs` as an alternative pattern
- **xUnit** — testing with `WebApplicationFactory`

## Getting Started

```bash
dotnet run
```

## Running Tests

```bash
dotnet test
```
