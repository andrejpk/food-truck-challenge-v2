# Java Spring Boot Starter

Starter shell for the Food Truck Challenge using Java and Spring Boot.

## Stack

- **Java 25** — latest JDK
- **Spring Boot 3.5** — REST API framework
- **JUnit 5** — testing with `MockMvc`
- **Maven** — build and dependency management

## Getting Started

```bash
mvn spring-boot:run
```

## Running Tests

```bash
mvn test
```
